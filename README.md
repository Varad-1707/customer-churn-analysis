# Customer Churn Analysis

## Objective
Analyze customer data to identify factors influencing churn.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Key Insights
- Contract type impacts churn
- Higher charges increase churn
- Long-term customers are more stable
